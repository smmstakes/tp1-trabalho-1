Descrição detalhada

Gerencia a conexão e o estado do banco de dados SQLite.

Implementado como um Singleton, esta classe garante que existe apenas um ponto de controle para o banco de dados em toda a aplicação. Ela abstrai os detalhes de baixo nível da abertura e fechamento da conexão.
