Descrição detalhada

Interface Abstrata para o Serviço de Autenticação de Conta.

Define todas as operações de autenticação que podem ser realizadas na primeira interação com o programa. As implementações desta interface conterão a lógica de validação e coordenação das operações.

const Senha & senha )
pure virtual

Permite inserir os dados para acessar a conta e os serviços do programa.

A implementação deve verificar regras de autenticação, como verificar se o CPF inserido existe no banco de dados e se a senha inserida é igual a senha associada ao CPF inserido.

Parâmetros
cpfO cpf do usuário, encapsulado em um objeto de domínio.
senhaA senha do usuário, encapsulado em um objeto de domínio.
Excepções
runtime_errorSe o CPF não for encontrado no banco de dados ou se a senha não corresponder à senha associada ao CPF inserido.

Permite inserir os dados para registrar uma conta no banco de dados.

A implementação deve verificar regras de autenticação, como verificar se o CPF inserido já existe no banco de dados antes de solicitar a criação na camada de persistência.

Parâmetros
cpfO cpf do usuário, encapsulado em um objeto de domínio.
nomeO nome do usuário, encapsulado em um objeto de domínio.
senhaA senha do usuário, encapsulado em um objeto de domínio.
Excepções
runtime_errorSe o CPF inserido já existir no banco de dados.
