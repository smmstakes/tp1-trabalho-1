Descrição detalhada

Interface Abstrata para o Repositório da entidade Conta.

Define o conjunto de operações de persistência de dados (CRUD) que podem ser realizadas com as contas. Por meio dessa interface o usuário é capaz de editar nome e senha, ler dados da conta e exclui-la
