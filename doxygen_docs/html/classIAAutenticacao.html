Descrição detalhada

Interface Abstrata para o Controlador de Apresentação da Autenticação de Conta.

Define o conjunto de operações que uma interface de usuário para autenticação de conta deve ser capaz de realizar. As classes que herdam de IAAutenticacao sao responsáveis por implementar a lógica de integração com o usuário.
