void Senha::validar ( const std::string & senha)
overrideprivatevirtual

Valida a senha conforme especificações.

A senha deve conter:

  • Seis caracteres;
  • Digitos alfanuméricos e os símbolos: #, $, % e &;
  • Nenhum caractere duplicado;
  • Pelo menos um dígito;
  • Pelo menos uma letra maiúscula e uma minúscula;
  • Pelo menos um caractere especial (#, $, % ou &).
Parâmetros
senhaSenha a ser validada.
Excepções
std::invalid_argumentSe a senha não atender aos critérios pedidos.
